# Silent Bridge

Full-stack healthcare accessibility platform developed in Expo (React Native) for matching Deaf users with suitable sign language interpreters for medical appointments based on availability. Features an appointment management system using Supabase to support bookings, interpreter scheduling, and secure in-app messaging with attachments to ensure sensitive personal information remains safely within the platform. Integrated with Google APIs to provide OAuth authentication for seamless onboarding, Google Calendar synchronisation for appointment reminders, and Google Meet video conferencing for clear sign language communication.

## Get started

1. Install project dependencies

   ```bash
   npm install
   ```

2. Start the application

   ```bash
   npx expo start
   ```

The output contains the following options to open the app:

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
